Juvenile History 1.0
Service Specification

Summary

The Juvenile History Service will be used by to retrieve history information (e.g. case plans, hearings, intakes, offenses, placement, referrals) associated with a juvenile in the state of Michigan.

Description

This service is intended to answer the question of "What information is available about a juvenile in Michiganâ€™s juvenile justice system?" It should enable service consumers to exchange information about the juvenile, reduce redundant assessments, and inform juvenile case planning.
